Project Name: Irrigation IoT Devices
This project aims to create a system that utilizes IoT devices to automate the irrigation process. The system will include simulated sensors to measure soil moisture levels and weather temperature to determine when and how much to water the plants.

How to Use?
1.Run web_server.py
2.Run EnCons_API.py
3.Run sensors.py
4.Run controller.py
5.Run To_TS.py
